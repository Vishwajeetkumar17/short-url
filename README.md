# short-url

## Initialize your project as

npm init -y

## Install these packages

1. npm i express
2. npm i mongoose
3. npm i nodemon
4. npm i shortid
5. npm i cookie-parser
6. npm i dotenv
7. npm i ejs
8. npm i jsonwebtoken
9. npm i uuid

## Add .env file

PORT=8001

MONGODB_URI=mongodb_uri

JWT_SECRET=generate_jwt_token

NODE_ENV=development

COOKIE_EXPIRY=7

BASE_URL=http://localhost:8001

## Run Command

npm start
